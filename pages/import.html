Import Produk dari Sarqu.id (Shopee)
  1. Buka https://shopee.co.id/sarqu.id#product_list, salin nama produk & harga, lalu bentuk JSON seperti contoh di bawah.
  2. Tempel JSON ke kotak, lalu klik Simpan Produk. Data disimpan di browser (localStorage).
  3. Alternatif cepat: klik Muat Seed untuk contoh produk.

Catatan: karena pembatasan CORS & login Shopee, website ini tidak bisa mengambil data langsung. Gunakan import manual ini.
